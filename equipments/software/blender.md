## 3. Shortcuts for 3D Modeling

Here is a summary of useful Blender shortcuts:

| Shortcut | Action |
|-----------|---------|
| `A` | Select all |
| `E` | Extrude |
| `S` | Scale |
| `M` | Merge |
| `7` | Top view |
| `1` | Side view |
| `G` + `Z` | Grab and move in *Z* axis |
